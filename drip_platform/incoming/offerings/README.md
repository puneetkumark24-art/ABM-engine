| Column | Required | Notes |
|---|---|---|
| name | yes | Product name, e.g. "Vahana" |
| category | | Platform / Module / Service |
| description | | |
| target_segments | | comma-separated, e.g. "Commercial Bank, Digital Bank" |
| key_benefits | | |
| competitors | | comma-separated, e.g. "Temenos, Mambu" |

Re-running the import is safe — a product already loaded (same name) gets
updated, not duplicated.

If you'd rather hand me a slide deck, PDF, or write-up instead of a
spreadsheet, that's fine too — drop it here and tell me, and I'll extract it
into this format for you.
